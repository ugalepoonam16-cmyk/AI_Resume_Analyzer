# 🤖 AI Resume Analyzer

An AI-powered Resume Analyzer that evaluates resumes, extracts skills, provides AI feedback, generates a resume score, and suggests improvements.

🌐 **Live Demo:** https://ai-resume-analyzer-six-hazel.vercel.app/

---

## 📌 Features

- 📄 Upload Resume (PDF, DOC, DOCX)
- 🤖 AI Resume Analysis
- 🎯 Resume Score (0–100)
- 💼 Skill Extraction
- 💡 Resume Improvement Suggestions
- 📝 AI Resume Feedback
- 📥 Download Resume Analysis Report (PDF)
- 📜 Resume History
- 🔍 Search Previous Reports
- 🗑 Delete Resume Reports
- 📱 Responsive UI
- 🔔 Toast Notifications
- ⏳ Loading Indicator

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/ugalepoonam16-cmyk/AI_Resume_Analyzer.git
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
node server.js
```
